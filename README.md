# Figure-generation-for-methane-metabolism-associated-transcriptomic-data

In this R file figures are generated to gain insight into the changes in methane metabolism associated genes. The microbe being studied is _Methylotuvimicrobium alcaliphilum 20ZR_ and the transcriptomic data details how this microbe responds to dynamic metal conditions. The figures generated in this R code are heatmaps and descriptive bar plots.
